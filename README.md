mang
====

Mang is a neural network library for python 2.x.
It requires the packages below:

* numpy (>= 1.6.2)
* easydict (>= 1.4)
* cudamat and gnumpy (https://code.google.com/p/cudamat/)

Please note that Mang uses a very slightly customized version of gnumpy.
